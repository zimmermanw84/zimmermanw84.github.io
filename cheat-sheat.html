Ruby Enumerable Cheat Sheet

Enumerables

Enumerable is a Ruby module. Enumerable provides a set of methods to a Class that includes it to be sorted, searched, and manipulated. For a class to be eligible to include Enumerable, the objects of the class (the things created by the class and that belong to the class) must be accessible via #Each. The string of characters, "abcdef"is not accessible with each:

"abcdef".each #=> NoMethodError: undefined method 'each' for "abcdef":String

However it is accessible via each_char:

"abcdef".each_char

The range of numbers 1-3 is accessible via each:

(1..3).each {|i| print i} #=>1;2;3;

Enumerable also allows the object of a class that includes it to be searched, sorted, and selected with methods like :find, :sort, and :select.

Enumerable Methods

each calls a block once for each element in a collection, passing each element to the block.

(1..3).each {|i| p i} #=> 1; 2; 3;

inject is like the cute and small version of Each. It combines all of the elements of an enumerable by passing it to a block and outputting the result.

(4..10).inject { |sum, n| sum + n } #=> 49

include? looks through each element in an enumerable and checks if it matches the paramater given.

(4..10).include?(6) #=> True

first returns the first element of an enumerable, if the enumerable is empty, it returns nil.

(4..10).first #=> 4

sort returns the elements of an enumerable encapsulated in an array and sorted either by their numerical value or sorted by the supplied block.

[5, 4, 3, 7, 12, 6].sort #=> [3, 4, 5, 6, 7, 12]

find passes each entry in enum to block. Returns the first for which block is not false. If no object matches, calls ifnone and returns its result when it is specified, or returns nil otherwise.

(1..3).find(2) #=> 2

map returns a new array with the results of running block once for every element in enum. Very silimar to each.

(1..3).map { |x| x + 1 } #=> [2, 3, 4]
