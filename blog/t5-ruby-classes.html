The Class System

Phase 0. Week 5 - Saturday, November 29th, 2014

Welcome back to my blog! This post is going to be covering Class's, instance variables, and instance methods. Below is an example of a class being created and under the initialize method, an instance variable is created.

Above you see how a class is created in ruby. It's very similar to the creation of a standard method. Nested within the class of "House" you have an instance variable named address. You can tell the difference between regular variables and instance variables with the @ symbol. We have @address set to a string, under initialize. This means that anytime a new house object is created it will start with @address set to "123 First st.".

So now we're are creating methods within the class. If you notice the interpolation in the string, we're using the instance variable. This is the reason why we would use an instance variable. If we set this variable for the class, we can call that variable through out all methods created within the class. This allows for a lot more versatility for your programs. Not having to set and create the same variable over and over again. Now here is the whole program.

Now if you look at the bottom it shows us creating a new House object called my_house. Then we are calling our class instance methods on my_house.

And there you have it. Us calling our methods on our my_house object gives us the output we expected.
